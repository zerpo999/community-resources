# Capstone Project
# 📌 Community Resource App

A web application that helps users find local community resources such as food banks, shelters, healthcare services, and other support organizations.

This project was built as part of a software learning experience using **React + Vite** and focuses on building real-world UI, search functionality, and structured resource data.

---

## 🚀 Features

- 🔎 Search resources by name or category  
- 🧭 Simple and clean navigation  
- 🖼 Logo and styled home page  
- 📱 Responsive layout (basic)  
- ⚛️ Built with modern React components

---

## 🛠 Tech Stack

- React  
- Vite  
- JavaScript  
- CSS / Tailwind (edit if needed)  
- Git & GitHub

## ▶️ Getting Started

Clone the repo:

```bash
git clone https://github.com/zerpo999/community-resource-finder.git
cd community-resource-finder
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the app in your browser (usually http://localhost:5173).
